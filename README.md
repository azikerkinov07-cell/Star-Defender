# 🌌 Star Defender

A classic arcade space shooter built with **Pygame**.

## ✨ Features

- Intense space combat
- 3 types of enemies + powerful **Bosses** every 5 levels
- Power-up system (Double Shot, Shield, Speed Boost)
- Dynamically increasing difficulty
- Beautiful explosion particles
- Clean menu with controls display

## 🎮 Controls

| Key                  | Action                      |
|----------------------|-----------------------------|
| `← →` or `A D`       | Move ship                   |
| `SPACE` (hold)       | Shoot                       |
| `ENTER`              | Start Game                  |
| `Q`                  | Quit                        |
| `R`                  | Return to Menu (after death)|

**Power-ups:**
- **Yellow** — Double Shot
- **Green**  — Shield (8 seconds)
- **Cyan**   — Speed Boost
